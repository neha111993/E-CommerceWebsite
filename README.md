# E-CommerceWebsite
a)	Developed a functional single page e-commerce application featuring a dynamic product catalog and side-drawer shopping cart.
b)	Used JavaScript arrays to handle CRUD operations.
c)	Implemented Data Persistence via the web storage API for data to remains intact across browser sessions and refreshes.
d)	Designed a real time Price Calculation engine that automatically updates subtotals as per the quantity and price modifiers.
